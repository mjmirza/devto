# devto

Public archive of cover banners for Mirza Iqbal's dev.to articles.

Each banner is generated locally by an adaptive design system, then committed
here so its raw URL can serve as the article cover image on dev.to.

## What is in here

- `banners/` rendered cover and figure images, named by article.

## How a cover URL looks

```
https://raw.githubusercontent.com/mjmirza/devto/main/banners/<name>.png
```

## Design rules every banner follows

- Mirza Iqbal brand only. Made and researched in Germany.
- Red brand family only. No blue, purple, or rainbow accents.
- Playfair Display and Work Sans. No Inter, Roboto, or system fonts.
- Light mode. No dark backgrounds.
- No edge bars or bands on any side.
- A design guard blocks any of the above before a banner renders.
